# Billentyűparancsok

Az alábbi táblázatban látható az összes elérhető billentyűparancs és azok függvénye. **Mindegyik szokás szerint beállítható** a Beállítások menüben.

## Értékelés

| Billentyű | Funkció | Megjegyzés |
|---|---|---|
| `1` – `5` | Csillagos értékelés (1–5) | Újra megnyomva ugyanaz a gomb törli az értékelést |
| `0` | Értékelés törlése | Bármelyik képnél működik |

## Jelölés és elutasítás

| Billentyű | Funkció | Megjegyzés |
|---|---|---|
| `Y` | Zászló (jelölés) be/ki | A jelölt képek zöld kerettel jelennek meg |
| `X` | Elutasítás be/ki | Az elutasított képek halványítva jelennek meg |

## Navigáció

| Billentyű | Funkció | Helyzet |
|---|---|---|
| `←` (balnyíl) | Előző kép | Teljes képernyős nézetben |
| `→` (jobbnyíl) | Következő kép | Teljes képernyős nézetben |
| `Esc` | Nézet bezárása | Teljes képernyős nézetből vissza a rácsnézetbe |

## Zoom (teljes képernyős nézetben)

| Billentyű | Funkció |
|---|---|
| Görgő fel/le | Zoom be/ki |
| `+` | Fokozatos zoom be |
| `-` | Fokozatos zoom ki |
| `Szóköz` | Zoom toggle (be/ki vált) |
| Dupla kattintás | Eredeti méretbe zoom vagy 3x nagyítás |

## Testreszabás

A billentyűparancsok módosítása:

1. Kattints a **Beállítások** (⚙️) gombra a fejlécben
2. A **Billentyűparancsok** szekcióban látod az összes értékelhető parancsot
3. Kattints a mező melletti input mezőre, majd nyomd meg a kívánt billentyűt
4. Kattints a **Mentés** gombra

> Az **Alaphelyzet** gombbal visszaállíthatod az eredeti billentyűket.
